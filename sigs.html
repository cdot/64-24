Help for mail tools
  1. Click "Copy HTML to clipboard"
  2. Open Thunderbird
  3. In the left bar, left-click to select the account you are adding the signature for.
  4. Right-click and select "Settings"
  5. Make sure "Use HTML" is selected.
  6. Paste the html into the "Signature text" box.
For Outlook on Windows:
  1. Use "Method 2" as described here.
For Outlook web:
  1. Using the mouse, select the rendered signature above and Ctrl+C
  2. Open Outlook Web
  3. Click the gear icon in the top right
  4. Click "Account"
  5. Click "Signature"
  6. Select "Add signature"
  7. Enter a signature name
  8. In the text box, Ctrl+V
Instructions for GMail used in a web browser (desktop)
  1. Click "Copy HTML to clipboard"
  2. Open GMail
  3. Click on the gear icon
  4. Click "See all settings"
  5. Scroll down until you find "Signature"
  6. Click "Create new"
  7. Paste into the right hand edit box
  8. If you want, select the signature name in "Signature defaults"
  9. Scroll down to the bottom and click "Save changes"
